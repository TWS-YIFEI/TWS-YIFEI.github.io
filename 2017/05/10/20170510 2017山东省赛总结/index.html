2017山东省赛总结

　　今年的省赛已经结束，这意味着我的ACM之路要告一段落了。开始复习考研，没有很多时间用来A题了。　 距离上次省赛已经有一年时间，上次省赛我还是一个什么都不会的弱菜。而这次省赛，我仍然是一个弱菜~~(>_<)~~
　　这次省赛是带着遗憾离开的。。。
　　但幸运的是，我的ACM之路一定还没有结束。 　　这次省赛是在青岛科技大学，需要坐一晚上火车才能到，于是我带了一本书来充实一下这次旅程，是俞敏洪的<<在绝望中寻找希望>>，结果最后就真的要绝望了O(∩_∩)O~
　　我们三个人，一位是马上要毕业了的大四师哥，另一位是我的舍友。
　　在第一天热身赛的时候感觉状态还可以，一人看一题，虽然最后卡了一题，但感觉发挥的还算可以。
　　第二天正式赛。刚上来是经验丰富的师哥发现I题比较水，稍微一推找到了规律，一个斐波那契数列，问第n位mod2是多少，那么就只判断这个数是不是奇数。然后发现只要是三的倍数就输出0，其余输出1。这时候看到n<=10^1000的，光想到这个数挺大了，晕晕的要去用Java敲了，忘记是3的倍数的数有一个特点，每一位数字之和也一定是3的倍数。然后看另一个G题也挺简单，是裸地计算i的m次方再求和的，于是我又去用java敲，结果怎么都不对。。。遇到这套题第一个坑。。。卡了将近一个小时，师哥又去用c敲，仍然不对。心态快要没了的时候，师哥想起刚开始有过一次提示，题目上的1e9印成1e8。改后a了。这时候晓鹏看J题，题意不明确，叫我来看，看了一会，终于明白题意，有很多货物，第i天卖出去就收益i×单价，问最大收益是多少。我想了一个暴力，感觉可以通过，于是开是敲，交了几发发现都不对，然后晓鹏又去敲，发现了另一种时间复杂度更低的方法，敲完提交，仍然不对。。于是在这个题也卡了好久。这时候时间已经过去三个多小时了，看了一下榜，感觉要再做两题才有把握在铜牌区。这时候有重新审视了一下J题，发现数据可能会超int。。。改longlong后a了。这时候好像已经封榜了，虽然又做了一个题，但感觉我们这时候已经没有希望了，然后有点泄气。继续看另一个题意超级简单地F题，遇到第二个大坑，following statement is true :“For any x, if a×x^2+b×x+c=0,then x is an integer.”看似挺简单的题意，却就是一直wrong，我们理解的是如果输入的a，b，c满足这个式子，就输出YES。那么如果delta<0就不满足这个式子，然后应该输出NO。然后怀疑自己智商到比赛结束。赛后问别人，题木意思是问这句话是不是对的，而不是输入的abc是不是对的。如果delta<0，题目中的第一个if就不满足，那句话仍然是对的，应该输出YES（F题赛后补题时，在理解了题意的情况下又叫了十几发仍然不对！）。在最后一小时我们又看了C和D，C是杨辉三角的，不过数据特别大，肯定没法用二维数组存储。然后想用一维数组，向树状数组那样存储，但是最后没有想好下标之间的规律，并且细节也特别多，写出来也不一定对。于是没写。然后晓鹏去尝试D题，然后提交n次仍然不对。
　　最后三题收场。确实感觉题目是挺坑的，一题印错数据，一题疑似玩脑筋急转弯。题目总体偏向数学及推理。
　　但是话又说回来，还是自己菜，同样的坑，为什么别人就可以A掉。 　　由于这又一次珍贵的打铁，我们才又发现了自己的很多缺点。我们数学题做的太少，思维有点打不开，以后应当对此进行一些专项训练。英语水平不够，像那个F题，实际上想说的是高中学的那种充分条件，if。。。then。。。对题意的把握不准确。革命尚未成功，英语还需加强！完整的参加比赛的次数少，心态容易受影响，卡题导致其他可以做出来的题都没有时间去看。
　　另外比完赛发现印的题目目录上有四个题目是粗体，不知道是什么鬼。。。
　　每次失败的机会都是很珍贵的，打铁的机会也是很珍贵的，可能从下一次就不再打铁了呢！
　　最近看过一句话，说搞竞赛的，每一次失败可能都会导致竞赛生涯的结束。确实每一次失败都会带给自己很大的打击，从上次参加省赛打铁，到如今一年了，进步了不少，但仍然在打铁。这样的情况总会让人有一些无力感。
　　比完赛回到学校，把《在绝望中寻找希望》看完了，改写书中的一句话来勉励自己，以及所有经受着失败的人：没有失败过的生命显得轻浮而没有分量，生命中的失败和摆脱失败的努力，使我们的生命变得厚重而美丽。
